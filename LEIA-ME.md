# EstudaAí — Plataforma de Estudos para Concursos

## Como instalar (uma única vez)

1. Extraia o ZIP em qualquer pasta (ex: `C:\Projetos\estudaai`)
2. Clique duas vezes no arquivo **INSTALAR.bat**
3. Aguarde instalar (2-3 minutos)
4. Acesse **http://localhost:3000** no navegador
5. Crie sua conta e comece a usar!

## Como usar depois

Abra o terminal na pasta do projeto e rode:
```
npm run dev
```

## Funcionalidades

- ✅ Login e cadastro de conta
- ✅ Dashboard com estatísticas em tempo real
- ✅ Sessão de Estudo com cronômetro real
- ✅ Ciclo Inteligente (fila automática de matérias)
- ✅ Matérias, Tópicos e PDFs
- ✅ Revisões espaçadas (24h, 7d, 30d)
- ✅ Flashcards com repetição espaçada (método Anki)
- ✅ Caderno de Erros
- ✅ Estatísticas com gráficos
- ✅ Planejamento semanal

## Banco de dados

Usando o PostgreSQL no Railway que você já tem configurado.
Os dados são salvos automaticamente.
